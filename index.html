Kubernetes Cluster on AWS.

When learning something new I like to start with the complete picture at hand. I may not understand what I am looking at, but having the full picture; then learning its individual parts is more enjoyable than the oposite, learning the parts then putting them together at the end. In this short guide I will show you how to create a Docker image from your code, deploy a Kubernetes cluster in AWS using EKS, and then run your "web app" in the Kubernetes cluster. If you're on a budget like me you will want to bring up your Kubernetes cluster mess around with it and then remove all the resources so you don't run up your AWS bill.

To follow along you will need the following. I am doing this on Ubuntu 22.04 but the same can be achived on Windows. These are the steps we will follow:

  • Create Docker Image
  • Create EKS cluster
  • Deploy the application on the EKS cluster

    Prerequisites:

    • eksctl
    • kubectl
    • Docker hub account

If you don't have any code of your own you can clone my repository:
